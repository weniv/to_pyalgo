# 조건 덧셈
## 문제 설명
주어진 리스트에서 3과 5의 배수를 제외한 모든 수를 더하는 solution 함수를 완성해주세요.

---

## 제한 사항

- 1 ≤ 리스트 안의 수 ≤ 100
- 숫자 외에 다른 값은 주어지지 않습니다.
- 각각의 값은 1차원 리스트로 주어집니다.
- 리스트 안에 더할 값이 없는 경우 0으로 출력해야 합니다.

---

## 입출력 예

|   입력    | 출력 |
| --------- | ------ |
| [1, 2, 3, 4, 5] | 7 |
| [3, 5, 7, 1] | 8 |

---

## 입출력 설명
[1, 2, 3, 4, 5]에서 3과 5의 배수를 제외하면 [1, 2, 4]가 됩니다. 이 수를 모두 더하면 7이 됩니다.
